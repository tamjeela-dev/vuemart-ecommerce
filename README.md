# 🛒 VueMart - E-commerce Website

A responsive e-commerce web application built using Vue.js and Vite.

## 🚀 Live Demo

👉 https://vuemart-ecommerce.vercel.app

## ✨ Features

- 🏠 Home Page
- 📦 Product Listing
- 🔍 Product Search
- 📄 Product Details
- 🛒 Shopping Cart
- 💳 Checkout Page
- ✅ Order Success Page

## 🛠️ Tech Stack

- Vue.js 3
- Vue Router
- Pinia
- Axios
- Bootstrap 5
- Vite
- DummyJSON API

## 📂 Installation

```bash
git clone https://github.com/tamjeela-dev/vuemart-ecommerce.git

cd vuemart-ecommerce

npm install

npm run dev
```
